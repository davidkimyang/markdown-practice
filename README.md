# Jordan Legacy (Next.js)

마이클 조던 팬 아카이브/SNS 콘셉트의 학습용 웹사이트입니다.

## 포함된 섹션

- Hero (Jordan Legacy)
- Career Timeline
- Gallery
- Quotes
- Free Board (닉네임 기반, 로그인 없음 / localStorage 저장)

## 실행

```bash
npm install
npm run dev
```

## 참고

- 현재 `public/images`의 이미지는 학습용 플레이스홀더 SVG입니다.
- 실제 공개 서비스에서는 이미지/로고/상표권 라이선스를 반드시 확인하세요.
